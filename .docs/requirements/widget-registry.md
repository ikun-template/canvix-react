# 组件注册

## 定位

当前阶段不实现外部扩展系统。所有组件为预设的内置组件，存放在 `domains/widgets/*` 中。但注册机制需保留，为后续扩展能力预留接口。

---

## 要求

### 组件定义

- 每个内置组件为独立模块，包含：组件元信息、渲染实现、属性面板配置、默认 Schema 预设值
- 组件元信息包含：唯一类型标识、显示名称、分类、图标、默认尺寸

### 注册机制

- 提供统一的组件注册表，所有可用组件需注册后方可使用
- 注册表支持按类型标识查询组件定义
- 注册在应用初始化阶段完成，运行时注册表内容不变

### 组件实例化

- 从 Toolbox 添加组件时，根据组件类型标识从注册表获取默认预设值，生成新的 Widget Schema 实例
- 画布渲染时，根据 Widget Schema 中的类型标识从注册表获取渲染实现

### 预留扩展

- 注册接口不限定组件来源，后续可扩展为支持远程/动态加载的外部组件
