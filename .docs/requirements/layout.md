# 布局系统

编辑器 UI 划分为四个布局区域，每个区域为独立模块。

---

## 布局区域

| 区域      | 职责                               |
| --------- | ---------------------------------- |
| Canvas    | 画布渲染、组件编排、拖拽/缩放/框选 |
| Sidebar   | 页面列表、图层树、资源管理         |
| Inspector | 选中对象的属性编辑面板             |
| Toolbox   | 可用组件列表，拖入画布添加         |

## 要求

- 每个布局区域为独立的插件模块，通过 Dock 的插件机制注册和挂载
- 布局区域之间不直接依赖，通过事件总线或 SDK 通信
- 布局区域支持按需加载，未注册的区域不影响其他区域运行
- 布局区域的显示/隐藏、尺寸调整由 Dock 容器统一管理
